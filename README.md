# VacumBook 📚

> **Версия:** 1.0  
> **Парсер книг с сайта** [labirint.ru](https://www.labirint.ru)

VacumBook — это простой парсер на Python, который собирает информацию о книгах с интернет-магазина «Лабиринт»: название, автора, цену и ссылку на товар, с сохранением данных в локальную SQLite-базу.

---

## ⚡ Особенности

- ✅ Сбор данных с первых 15 страниц каталога книг
- ✅ Парсинг: название, автор, цена, ссылка
- ✅ Сохранение в SQLite-базу с защитой от дубликатов (`INSERT OR IGNORE`)
- ✅ Автоматическая очистка данных (удаление пробелов, символов валюты)
- ✅ Задержка между запросами (4 сек) для снижения нагрузки на сервер
- ✅ Логирование процесса парсинга в консоль

---

## 📦 Требования

- Python 3.7+
- Установленные зависимости:

```txt
requests
beautifulsoup4
lxml
```

## 🚀 Установка и запуск

1. **Клонируйте репозиторий** (или скачайте код):
```bash
git clone https://github.com/your-username/VacumBook.git
cd VacumBook
```

2. **Создайте виртуальное окружение** (рекомендуется):
```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
```

3. **Установите зависимости**:
```bash
pip install requests beautifulsoup4 lxml
```

4. **Подготовьте структуру папок**:
```bash
mkdir -p database
```

5. **Запустите парсер**:
```bash
python main.py
```

> 📁 База данных будет создана автоматически по пути `./database/database.db`

## 🗄️ Структура базы данных

**Таблица:** `ALL_BOOKS`

| Поле | Тип | Описание |
|------|-----|----------|
| `TITLE` | TEXT | Название книги |
| `PRICE` | INTEGER | Цена в рублях |
| `LINK` | TEXT | Ссылка на страницу книги |
| `AUTHOR` | TEXT | Автор книги (или "Author not found") |

> ⚠️ Используется `INSERT OR IGNORE` — дубликаты по уникальным значениям не добавляются.

---

## ⚙️ Настройки парсера

| Параметр | Значение по умолчанию | Описание |
|----------|----------------------|----------|
| `страниц для парсинга` | 1–15 | Диапазон страниц каталога |
| `задержка между запросами` | 4 сек | `time.sleep(4)` для этичного парсинга |
| `путь к БД` | `./database/database.db` | Локальный SQLite-файл |

> 💡 Чтобы изменить диапазон страниц, отредактируйте цикл:  
> `for page in range(1, 16):` → `range(1, N+1)`

---

## 📤 Пример использования данных

```python
import sqlite3

conn = sqlite3.connect('./database/database.db')
cur = conn.cursor()

# Получить все книги дешевле 500 ₽
cur.execute("SELECT TITLE, AUTHOR, PRICE FROM ALL_BOOKS WHERE PRICE < 500")
for row in cur.fetchall():
    print(f"{row[0]} — {row[1]} ({row[2]} ₽)")

conn.close()
```

## ⚠️ Важные замечания

1. **Соблюдайте правила сайта**  
   Перед использованием парсера ознакомьтесь с [условиями использования labirint.ru](https://www.labirint.ru/about/terms/). Массовый сбор данных может нарушать их политику.

2. **Не увеличивайте частоту запросов**  
   Задержка в 4 секунды — минимальная рекомендация. Уменьшение интервала может привести к блокировке по IP.

3. **Структура сайта может измениться**  
   Если парсер перестал работать — вероятно, изменились CSS-классы на сайте. Проверьте актуальные селекторы в инспекторе браузера.

4. **Обработка ошибок**  
   В текущей версии при ошибке сети парсинг продолжается со следующей страницы. Для продакшена рекомендуется добавить повторные попытки и логирование.
